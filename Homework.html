Mac/linux 注意事项

安装Python, 为了稳定性，请让python的版本高于3.5以上

推荐你们把.py 文件放到桌面上，这样易于找到你们的作业（因为作业文件夹和.py文件都在同一父目录）

通过terminal(终端） cd 到.py文件所在的目录，再执行下一步。

作业生成指令： python Student.py

以下是部分mac 的terminal 指令

如果是放在桌面 -> cd desktop

如果想查询下级菜单 -> ls

返回上级菜单 -> cd ..
